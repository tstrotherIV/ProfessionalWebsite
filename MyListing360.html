MyListing360 VR React App

  • Built with React
  • 2020

App Overview

MyListing360 is a resource for Real Estate Agents and Brokerages as a response to the COVID Pandemic. Fewer Agents and Buyers are willing to visit homes in person due to restrictions so Agents have tried to come up with creative Solutions for showing listings. MyListing360 is a great simple solution to this problem by providing a resource for Agents to be build and share immersive Virtual Tours in only minutes using 360 images of the home.

In order to create the Virtual Tours a Dashboard was created. This User dashboard will show all tours built by that user. Also within the Dashboard is an area to creat new tours which will then poplulate in the Dashboard.

Technology Used:


•ReactJS via "Create React App"
•A-Frame VR (Three.js and WebVr.js) to display images in a Virtual Environment
•Cloudinary for image hosting
•React Bootstrap for styling
•React Google Analytics for User data tracking
•HTML/CSS/JS
